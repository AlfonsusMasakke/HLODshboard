## ✨ Features

- 📊 **Real-time Analytics Dashboard**
- 📈 **Interactive Revenue Charts** 
- 🏢 **Partner Management System**
- 📋 **Monthly Detail Reports**
- 📤 **Excel Export Functionality**
- 🔐 **JWT Authentication**
- 📱 **Responsive Design**

## 🚀 Technology Stack

**Frontend:**
- Vue.js 3 + Composition API
- Pinia (State Management)
- Chart.js (Data Visualization)
- Axios (HTTP Client)

**Backend:**
- Node.js + Express.js
- MySQL + Sequelize ORM
- JWT Authentication
- RESTful API

## 📦 Installation

### Prerequisites
- Node.js >= 16.x
- MySQL >= 8.x
- npm atau yarn

### Setup Backend
```bash
cd backend
npm install
cp .env.example .env
# Configure your .env file
